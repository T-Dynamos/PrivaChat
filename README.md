## Install
Its simple, install python and following deps using pip
```python
kivy==2.1.0
kivymd=1.1.0
colorama
plyer
```
for gui run
```shell
python3 main.py
```
for cli run
```shell
python3 cli.py
```
For apks see releases
> Cli also works on termux
